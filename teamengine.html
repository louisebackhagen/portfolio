TeamEngine Website

Home Portfolio Project

User Research | Interactive Prototyping | Adobe XD | HTML & CSS | JavaScript

Challenge

When this project began, the company had just developed a new visual identity. My task was to implement this new graphical profile in the design of the upcoming website, ensuring consistency with the updated brand guidelines.

  • Determine what the main elements should look like based on the new graphical profile for the company.
  • Determine which functionalities to remove.
  • Determine which functionalities to enhanced.

Process

  • Conducted a focus group with the CEO, Customer Success team, and other users to gain valuable insights into how the existing website was used and to identify key areas for improvement.
  • Began the design phase with initial sketches exploring layout, navigation, and visual hierarchy based on user feedback and brand guidelines.
  • Held regular review sessions with both designers and stakeholders to refine concepts and ensure alignment with the company’s new visual identity.
  • Created interactive digital prototypes in Adobe XD CC to test usability and gather feedback from real users.
  • Iterated on the design based on insights from testing, focusing on improving clarity, accessibility, and overall user experience.
  • Developed the new website based on the finalized design, ensuring consistency with the updated graphical profile and optimized performance across devices.

Delivery & Impact

  • The new website was successfully launched, fully reflecting the company’s updated visual identity and creating a cohesive brand experience.
  • Improved usability and navigation made it easier for visitors to find key information, leading to higher engagement.
